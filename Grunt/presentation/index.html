Instalacion & preparacion

Grunt instalation:

npm install -g grunt-cli

npm install grunt --save-dev

Shortcut: "npm i grunt -D" (-S == --save)

Agregando Uglify:

npm i grunt-contrib-uglify --D

Carpeta: node_modules

Ideal: No subir al repo, ejecutar: "npm i"

Gruntfile.js & package.json

Gruntfile.js: Configuracion de Grunt, con definicion de tareas.

Package.json: Configuracion de nodeJs, definicion de componentes/plugins instalados en el proyecto

Package.json (NodeJS)

			{
			    "name": "myJsProject",
			    "version": "0.0.0",
			    "description": "short description",
			    "author": "me",
			    "date": "2014-03-27 17:31:31 -03:00",
			    "devDependencies": {,
			        "grunt": "~0.4.2",
			        "grunt-contrib-uglify": "~0.2.7"
			    }
			}

Command: "npm init" wizard para la creacion de package.json

Gruntfile.js:

			module.exports = function(grunt) {
				grunt.initConfig({
					pkg: grunt.file.readJSON('package.json'),
					uglify: {
						options: {
							banner: '/*! <%= pkg.name %>'
						},
						build: {
							src: 'src/<%= pkg.name %>.js',
							dest: 'build/<%= pkg.name %>.min.js'
						}
					}
				});
				grunt.loadNpmTasks('grunt-contrib-uglify');
				grunt.registerTask('default', ['uglify']);
			};
